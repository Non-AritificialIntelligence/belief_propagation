# Belief Propagation: Cavity Method to Study Phase Transitions in Disorderd Systems

## Overview

A small project where the belief propagation is applied on a problem in statistical physics.

## Features

* **Implementation** of the belief propagation on a solid state material
* **Comparison** with results via direct diagonalization
* **Phase Transition** from a transistor to a conductor (coming soon)

## How to run

1. Install 'numpy', 'matplotlib.pyplot', 'networkx' and 'scipy'
2. run 'Disordered System.py'
